# Pica 仓库成品库 (Repository Registry)

本目录存放 CI 自动化生成并发布的**实时索引文件**。

## 📄 repo.json

这是仓库的“核心地图”。它包含了：
*   所有可用软件包的列表。
*   每个包的版本、架构信息。
*   包内容的 SHA256 哈希值（用于安全校验和智能缓存）。

## 🌐 分发说明

*   此文件夹的内容由 GitHub Actions 自动更新。
*   Cloudflare Worker 会从此处拉取 `repo.json` 并分发给全球各地的 Pica 客户端。

**警告：请勿手动修改此目录下的文件，因为它们在每次 CI 运行后都会被自动覆盖。**
